Tensorium Wallet Support

Practical help for installing, unlocking, backing up, and troubleshooting the Tensorium Wallet browser extension.

Common Problems

I forgot my password

If you still have the wallet JSON backup, you can import it again and set a new password. If you lost both the password and the backup, there is no recovery path.

The wallet says the node is unreachable

Check the selected network in Settings. The extension relies on an RPC endpoint to read UTXOs and broadcast transactions. If the default RPC is temporarily unavailable, try again later or switch to a trusted custom RPC.

My balance looks wrong

The current extension uses UTXOs reported by the configured RPC and shows mature balance. Recently mined testnet coinbase outputs may need confirmations before becoming spendable.

Why is history incomplete?

The current history view focuses on received transfers and does not yet include full outgoing detection. Explorer remains the best source for complete on-chain visibility until a dedicated indexer is added.

Backup Checklist

  • download the wallet JSON immediately after creation
  • keep at least one offline copy
  • do not send the private key over chat or email
  • test that your backup file can be imported before relying on it

What to Include in a Bug Report

  • browser version
  • whether you used testnet, mainnet-candidate, or custom RPC
  • exact action that failed
  • error message shown in the popup
  • steps to reproduce

Do not include private keys, raw wallet backups, or passwords in bug reports.
